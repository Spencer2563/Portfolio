Creative Expression 1.1.6:

Project: 1.1.6 Creative Expression
Course: Computer Science Essentials
Duration: Approximately 2 weeks (10/19/23-10/31/23)
Objective: Create an original app in MIT app inventor.
Group size: Group of 2

In this project my partner, Riri Bhutani, and I were tasked with creating an app from our own minds that related to our hobbies or interests. Throughout development of this app we utilized SCRUM to organize our progress into one neat document, that would be graded according to this rubric. Together we thought of making an app that would not only be fun but leave the user with a work of art once they were done. I was inspired by marble painting that I did as a child. Together we thought, what if we make this into a game where a trail is left by a snail that eats different fruit to change the color of its trail, all the while avoiding an enemy (the enemy can be disabled in settings if the user would like a more peaceful experience). During this project I learned the importance of teamwork and a shared goal. With enough time and dedication, I found, you can get anything done. I was impressed with the work I managed to get done in such a short period of time; All thanks to great organization.


Sandwich Project:

Project: 3.1.3, 3.2.1, and 3.2.3 Sandwich app
Course: Computer Science Essentials
Duration: Approximately 1 month
Objective: Create a console app in Python simulating an ordering system for sandwich shops.
Group size: Individual

In these projects we were tasked with creating a sandwich ordering program. While not offering total creative freedom, the way we implemented functionality was entirely up to us. The interesting thing about this project was that we iterated over the course of 3 projects, this allowed the opportunity to learn many new skills such as: implementing while loops and lists. Additionally this meant that documentation and tracking of the work was done in 3 Github Repositories. These projects were graded based on how well you implemented features for your sandwich app. This project allowed me to learn through doing. At the beginning I created a bare minimum working project, but in the end I had a project with a UX that updated to different sections (home, order, sandwiches, etc) and used a variety of functions and a while loop. Overall this project was a great learning experience and allowed me to learn through doing rather than reading a set of linear instructions.
